# territorial-nitrogen-flow-analysis

## Objective

This project aims to simulate nitrogen surpluses and deficits
between agricultural farms in order to explore crop-livestock complementarities
and nutrient circularity at territorial scale.

## Main Features

- Nitrogen balance calculation
- Surplus/deficit analysis
- Nutrient exchange simulation
- Territorial approach
- Data visualization

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Perspectives

This work contributes to the reflection on sustainable agriculture,
nutrient circularity and territorial agroecological transition.
